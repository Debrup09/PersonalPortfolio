# My Personal Portfolio

Welcome to the repository for my personal portfolio website. This project was built from scratch to showcase my skills, projects, and work experience in a clean, modern, and data-driven interface.

Live Demo: https://debrupsaha.netlify.app/

![20251027_145919-ezgif com-optimize](https://github.com/user-attachments/assets/b50746ef-c0ec-499d-8699-38a0a1ba0fdf)


## ‚ú® Features

* **Sleek UI:** A responsive, dark-mode-first design built with Tailwind CSS.

* **Component-Based:** Fully componentized using React, with reusable components for sections, cards, and icons.

* **Data-Driven Content:** All personal data (experience, projects, skills) is decoupled from the UI and managed in a single data.ts file, making updates incredibly simple.

* **Interactive Project Filtering:** Users can filter the projects list by technology with a single click, powered by the useState hook.

* **Strongly Typed:** Built entirely with TypeScript for better maintainability, scalability, and developer experience.

## üí° How to Update Your Portfolio

This portfolio is designed to be easily updated without touching the main React components.

- Navigate to `src/data.ts.`

- Find the section you want to update (e.g., workExperience, projects, skills).

- Add, edit, or remove items from the JavaScript arrays.

- Save the file, and the live development server will automatically update the UI with your new information!
